# midnight-tech theme

A dark theme for technical talks, code walkthroughs, and architecture reviews.

- Fonts: Inter + JetBrains Mono
- Background #0d1117 with #58a6ff (blue) and #7ee787 (green) accents
- `> ` prompt decoration before h1, code blocks have line numbers (CSS counter)

Mermaid or excalidraw is recommended for diagrams — they get rendered with colors that fit the dark background.
