
Marp를 활용한 Claude Code Plugin을 기획

기존의 문제는 Marp에서 쓸만한 Theme가 눈씻고 찾아보기 힘들다는 점이었음.
이를 클로드 코드의 디자인으로 해결한다는 1차적 목표

원래는 Marp를 그냥 스킬로 쓰게만 해주는 걸 생각했는데 기획에 살 덧붙이다보니 달라짐.

쓸만한 Theme를 몇 개 만들어서 저장시키고, 다른 사람의 테마도 넣거나, 필요하다면 '커맨드' 실행으로 원하는 테마를 만들게도 하는 것이 키 포인트

컨셉상으로는 "NotebookLM의 Slide 생성 기능을 Marp와 Claude Code로 달성해본다" 라고 생각하면 됨

Theme에 대하여

- marp에서의 Theme는 단순히 .css만 정의하지만, 나는 조금 더 좋은 슬라이드를 만들기 위해 "샘플" 슬라이드도 같이 제공되어 있어야 한다고 생각함
- 글로벌 사용을 의도하여, `sample.md` , `sample.en.md`, `sample.jp.md` 와 같이 여러 언어로 된 예시도 있으면 좋을듯
- 후술하겠지만 prerender config가 들어간 config.json 파일도 있으면 좋을 것 같음
- Theme 메타데이터도 있으면 좋을 듯함. 이것을 사용하는데 필요한 규칙(예, 어떤 태그는 css로 뭘 해놨기 때문에 어떤 제약이 있다) 혹은 "이 슬라이드는 기술적 발표에 특히 적합함" 등을 알리는 것
- 위에서 말한 것들은 하나의 디렉토리로 묶여서 관리된다.

Prerender에 대하여

- Marp 자체로는 슬라이드 생성이 썩 시원치 않음
- mermaid나 excalidraw 와 같은 애들의 도움을 받으면 좋음
  - 이 친구들은 .svg로의 전환이 가능하기 때문에, 먼저 .svg로 바꾸고 이것들을 .md에 지정시켜 결과적으로 .pdf나 .png에 담겨서 나오게 하는 것이 가능할 것이다.
  - 원본의 .md 파일에 일종의 placeholder (혹은 file 경로만 가르키는 링크) 를 넣어두고, 중간 prerender 단계에서 그것을 바꾸는 형식
- 클립 아트의 삽입을 Claude Code 도움을 받으면 좋음
  - 사용자가 Adobe photostock 같은 스톡 포토를 쓸 수 있는 API 서비스가 있다면, 그것을 세팅해서 Marp가 적절한 곳에 적절한 클립 아트를 찾아 넣을 수 있게 함
  - 사용자가 Gemini API 키가 있거나, Midjourney 키 등이 있다면, 이 또한 LLM이 알아서 호출까지 해서 적절한 이미지를 찾아 넣음
- Prerender는 새로운 유형의 매체와 API에 대응이 용이한, 확장 가능한 구조가 되어야 함

Project에 대하여

- Theme는 일종의 Template으로, 이 Theme를 이용해서 유저는 Slide(Marp Deck)을 만들어야 함.
- 프로젝트 또한 디렉토리 단위로 관리됨
- 프로젝트에서는 Theme의 override CSS를 사용할 수 있음. (테마에서 이 부분만 내가 손봐서 쓴다)
- 프로젝트에는 /build 디렉토리를 만들어서 최종결과물을 보관시킬 수 있음 (그러니 .gitignore에 등록하는 것을 권장)
- 프로젝트에서 어떻게 Deck을 생성시킬 것이냐. 몇가지 방법이 있음
  - 그냥 아이디어만 주절주절 클로드 코드한테 직접 말해서 생성시킨다
  - 프로젝트 내부에 blueprint 파일을 생성하고, 이것을 읽게 한다. (장문의 지시사항에 적합)
  - 슬라이드 생성에 필요한 자료들은 /assets 디렉토리에 저장한다. 이 때, 클로드 코드가 Fetch나 git clone 같은 것으로 외부로부터 자료를 수집해 Deck을 만드는데 참고시킬 수 있다.
  - prerender 되기 전의 결과물을 프로젝트에 output.md 로 중간 저장시킬 수 있다. -> 이것을 토대로 placeholder를 맞는 svg파일이나 img 파일로 변환한다.

인터페이스에 관하여

- 기본적으로는 slash command로 해결하도록 한다. (build 등)

Theme 관리에 관하여

- 아직 확정된 게 없음. 이 프로젝트를 만들때 이미 샘플 테마를 만들겠지만, 다른 유저가 만들었거나, 내가 추가적으로 만든 테마는 어떤식으로 추가 및 제거 등을 할 것인지 고민이 필요함
